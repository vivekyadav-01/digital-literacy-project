## Platforms Set Up

### 1. GitHub (github.com)
- **Purpose:** Code hosting, version control, and project showcase
- **What I did:** Created a profile README repository (special repo named after my username) with my name, branch, year, and learning goals
- **Why it matters:** GitHub is the global standard for developers. Employers and internship recruiters review GitHub profiles to assess a student's practical skills and projects.

### 2. LinkedIn (linkedin.com)
- **Purpose:** Professional networking, internship discovery, and industry connections
- **What I did:** Filled in the Education section with B.Tech CSE, VIT Bhopal University, and expected graduation year 2028
- **Why it matters:** LinkedIn is the world's largest professional network. Starting early means building connections and a visible profile before internship season begins.

## Reflection (for Project Report)

For Task 2, I set up professional profiles on two key platforms: GitHub and LinkedIn. 

GitHub is the world's leading platform for hosting and sharing code. As a Computer Science student, it will serve as my primary portfolio where I can showcase all projects, assignments, and personal experiments over the next four years. I created a profile README — a special repository that appears at the top of my GitHub profile — featuring my name, branch, year, and a personal learning goal statement.

LinkedIn is the most widely used professional networking platform globally. It is actively used by recruiters, hiring managers, and industry professionals. I set up my Education section with my current degree, institution (VIT Bhopal University), and expected graduation year of 2028. Keeping the profile minimal and professional at this stage is intentional — I will add internships, certifications, and skills progressively as I complete them.

Over the next four years, I plan to commit every significant project to GitHub so that my progress is visible and verifiable. On LinkedIn, I aim to connect with seniors, alumni, and industry professionals, and eventually document internships and achievements. Both platforms will together form the backbone of my professional digital identity long before I graduate.
